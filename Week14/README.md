# Week 14 - Dcard Clone

這是一個使用純 **HTML**、**CSS** 與 **JavaScript** 實作的 Dcard 網頁複刻版，模擬了 Dcard 的深色模式與核心互動功能。此專案為 Week 14 的實作練習，旨在建立一個單頁式應用 (SPA) 的基礎架構。

## 🎯 專案目標

複刻 Dcard 的核心使用體驗，包含瀏覽文章列表、篩選看板文章，以及查看文章詳情與留言。

## ✨ 功能特色

### 1. 介面與導覽 (UI/Navigation)
- **深色模式設計**：採用 Dcard 標誌性的深藍色系風格 (`#00324e` 等) 打造舒適的閱讀體驗。
- **全域導覽列 (Navbar)**：包含 Logo、搜尋欄與常用的功能圖標 (發文、通知、個人檔案等)。
- **側邊攔 (Sidebar)**：
  - 提供「所有看板」、「即時熱門看板」等導覽。
  - **看板篩選**：點擊特定看板 (如淡江大學、感情、研究所) 可篩選出該看板的文章。
  - **最近瀏覽**與**追蹤看板**清單。

### 2. 文章瀏覽 (Feed)
- **動態列表渲染**：使用 JavaScript 動態生成文章卡片。
- **分頁切換**：
  - **全部 (All)**：瀏覽所有類型的文章。
  - **追蹤 (Following)**：僅顯示已追蹤看板的文章。
- **文章卡片**：顯示看板名稱、發文者匿名/校系、發文時間、標題、摘要預覽及互動數據 (愛心、留言、收藏數)。

### 3. 文章詳情 (Post Details)
- **單頁切換體驗**：點擊文章卡片後，不換頁直接切換至詳情視圖。
- **完整內容展示**：顯示作者資訊、完整文章內文與圖片。
- **留言互動**:
  - **瀏覽留言**：模擬留言列表展示，包含樓層、校系與留言內容。
  - **新增留言**：使用者可於文章下方輸入文字並送出 (按 Enter)，系統會即時將新留言渲染至列表 (前端模擬)。
- **返回功能**：提供返回按鈕快速回到文章列表。

## 🛠️ 使用技術

- **HTML5**: 使用語意化標籤 (Semantic HTML) 建構網頁結構 (Header, Main, Aside)。
- **CSS3**:
  - **Flexbox & Grid**: 處理複雜的版面配置 (兩欄式佈局)。
  - **Responsive Layout**: 自適應容器設計。
  - **Styling**: 自定義 CSS 樣式與細節調整。
- **JavaScript (ES6+)**:
  - **DOM Manipulation**: 動態更新 DOM 元素以渲染文章與詳細資料。
  - **Mock Data**: 使用 JavaScript 物件陣列模擬後端資料 (文章、留言)。
  - **Event Handling**: 處理點擊、篩選與視圖切換邏輯。
- **外部資源**:
  - **FontAwesome**: 使用 `fa-solid`, `fa-regular` 等圖標庫美化介面。
  - **Picsum Photos**: 用於生成隨機的範例圖片與頭像。

## 🚀 如何執行

1. 將專案下載至本機。
2. 確保目錄中包含 `index.html`, `styles.css`, `script.js` 檔案。
3. 直接使用瀏覽器打開 `index.html` 即可開始瀏覽。
