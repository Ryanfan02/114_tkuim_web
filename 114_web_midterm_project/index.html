我們的設計建議

視覺層次

  • 引入微漸層與玻璃擬態（glass）效果，搭配陰影與懸浮 lift 動畫。
  • 使用 .lead、.small 與間距工具建立節奏。
  • 章節間加入圖塊（tiles）與徽章（badges）強調重點。

可用性

  • 深色模式時提高文字/背景對比，鏈結與重點元素換用高彩度色。
  • 表單採浮動標籤（floating labels），聚焦時顯示亮邊框。
  • 行動裝置上維持 16px 以上字級與舒適觸控距離。
